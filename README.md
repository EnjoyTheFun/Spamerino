# Spamerino

Chrome/Firefox extension for managing and pasting Twitch messages. Save, organize, and quickly insert your favorite messages using keyboard shortcuts and a fast search system. If you prefer not to rely on macros from third-party extensions or external apps, this browser extension provides a lightweight, in-page solution.

_This extension is still in testing and not yet published to official add-on stores._

## Features

### Core Features
- **In-Page Widget** - Use key combination or the icon under chat on any Twitch channel to open an overlay widget
- **Quick Paste** - Navigate with the arrow keys or click an entry in the panel to paste it directly into the chat input box
- **Smart Search** - Filter by content, `#tags`, or `channel:name`
- **Drag & Drop** - Reorder copypastas in the widget
- **Copy to Clipboard** - One-click copy button for each entry
- **Channel Scoping** - Limit copypastas to specific channels or make them global
- **Import/Export** - Backup and restore your collection as JSON
- **Privacy first** - All data is stored locally in the browser; nothing is sent to external servers

### Extension Popup
Click the Spamerino icon in your browser toolbar to access the settings:
- **Show chat button** - Toggle the chat button visibility in Twitch chat
- **Reset all data** - Permanently delete all saved copypastas
